# Machine_Learning_Wine_Project
A Machine Learning Wine Project predicts wine quality (e.g., as "good" or "bad" or on a 0-10 scale) based on physicochemical properties like acidity, alcohol content, pH, and sulfates. Using datasets like the Portuguese "Vinho Verde," models like Random Forest and XGBoost often achieve high accuracy (up to 100% in specific studies) for classifying.
